HOW TO PLAY

◆ CONTROLS

← → Arrow Keys / A D: Move paddle
SPACE: Launch ball / Resume game
ESCAPE or P: Pause game during play

◆ OBJECTIVE

Cooperate with your comrade to destroy all blocks.
Two paddles, one ball, shared lives!
Don't let the ball fall below the paddles.

◆ MULTIPLAYER

Create a room and share the code with a friend.
Or join using their room code.
Connection is P2P via WebRTC - no servers needed!

◆ ACCESSIBILITY

The game supports keyboard navigation with Tab and Enter keys.
Screen reader announcements are provided for game events.
Press Escape to quickly access the pause menu.
