# Arkanoid P2P — Кооперативный арканоид на WebRTC

Мультиплеерный арканоид без сервера, работает напрямую между двумя игроками через WebRTC DataChannel.

## Стиль: Гоп-стоп
- Яркие неоновые цвета на тёмном фоне
- Постсоветская эстетика 90-х
- Шрифты в стиле retro/vaporwave

## Архитектура
- **index.html** — основной HTML с Canvas
- **css/style.css** — стили в стиле Гоп-стоп
- **js/**:
  - `network.js` — WebRTC + signaling через код комнаты
  - `game.js` — игровая логика, физика, синхронизация
  - `levels.js` — генератор 20 уровней
  - `bonuses.js` — система бонусов
  - `ui.js` — UI экранов (меню, комната, игра)
  - `main.js` — точка входа, инициализация

## Как играть
1. Открыть игру в браузере
2. Создать комнату или ввести код
3. Поделиться кодом с другом
4. Играть!

## Технологии
- Vanilla JavaScript
- WebRTC (RTCPeerConnection + RTCDataChannel)
- HTML5 Canvas
- CSS Grid/Flexbox
